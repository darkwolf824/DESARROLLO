Contenido 1

La versión EE incluye todas las características de la versión CE más: Soporte de bases de datos NoSQL: MongoDB InfluxDB Apache Cassandra Redis Apache Hive Configuración de controladores adicionales para bases de datos BigData / Cloud (Hadoop). Los controladores JDBC más populares se incluyen en la distribución y se pueden usar en un entorno de acceso a Internet sin conexión / limitado. Complementos adicionales de EE: Generador de consultas de Visual SQL Generación de gráficos analíticos. Generador de datos simulado avanzado Soporte de formatos de oficina (XLS) para la exportación de datos Base de datos persistente del Administrador de consultas (permite realizar un seguimiento del historial de SQL) Visor / editor de formato JSON (en MongoDB y bases de datos relacionales) Procesador de gráficos de series temporales (para InfluxDB y otras bases de datos de series temporales) Eclipse Marketplace (permite instalar fácilmente complementos adicionales de Eclipse) Nos dará la posibilidad de apoyar y desarrollar la versión de la Comunidad, agregar nuevas funciones más rápido, brindar un mejor soporte y mucho más. Incluso si no necesita características empresariales, puede comprar una licencia como forma de donación. ¡Gracias!

Contenido 2

DBeaver es un cliente SQL y una herramienta de administración de base de datos. Para las bases de datos relacionales , utiliza la API de JDBC para interactuar con las bases de datos a través de un controlador JDBC. Para otras bases de datos ( NoSQL ) utiliza controladores de base de datos propietarios. Proporciona un editor que admite la finalización de código y el resaltado de sintaxis . Proporciona una arquitectura de complementos (basada en la arquitectura de complementos de Eclipse ) que permite modificar gran parte del comportamiento de la aplicación para proporcionar una funcionalidad de base de datos o características que son independientes de la base de datos. Esta es una aplicación de escritorio escrita en Java y basada en la plataforma Eclipse . La edición comunitaria de DBeaver es gratuita como software de código abierto que se distribuye bajo la Licencia de Apache . Una edición empresarial de código cerrado de DBeaver se distribuye bajo una licencia comercial.
